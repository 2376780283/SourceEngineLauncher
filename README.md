# Termux 终端构建完整教程  
> 基于 flutter aarch64 开发包 和 ndk r27b
---

- 需要准备一个 termux #linux aarch64_x64_x86 终端仿真器
- 需要 flutter ndk 社区 aarch64 的版本 x64_x86的设备直接跑proot x86_64 下载开发工具就行了
--- 到这一步 x64_x86 设备可以不用看了

- 按照流程配置工具 打开项目就行了
